In this Repo We will learn how to create Full stack web App using Django framework,
 javascript, jquery, HTML, and python.

Django Level One:

1- Install Virtual Environment pip3 install vertualenv

2- Create first project: django-admin startproject <project name>

3- Create first App : cd to your project then you'll find manage.py file run it and pass startapp parameter ex: "manage.py startapp <app name>"

4- Add your App Name to sitting.py file in INSTALLED_APPS  list

5- Also in sitting.py you need to add all paths for templates and static

In this level you should learn how to install Django on vertual enviroment (venv) and create venv, create Django project, Create Django App, Navigation with URLS (paths).
And Templates, Static files.
